 ## 📌 Project Overview

**Reserv** is a full-stack appointment scheduling web application designed to help users book services online and allow service providers to manage their services, availability, and appointments in one place.

The platform can be used by businesses and professionals such as hair salons, beauty studios, consultants, trainers, and other service-based providers who need a simple and organized way to handle bookings.

---

## ✨ Key Features

- User role selection between client and service provider
- Service registration and management
- Appointment booking flow
- Date and available time slot selection
- Working days and working hours configuration
- Appointment overview with status tracking
- Service filtering by service type and provider
- Edit and delete functionality for services and appointments
- Modern, clean, and responsive dashboard interface

## 🎯 Project Purpose

The goal of this project is to build a real-world appointment booking system with a modern user interface, structured backend architecture, and practical business logic.

The project demonstrates full-stack development skills, including frontend application structure, REST API communication, database persistence, authentication, role-based flows, and appointment scheduling logic.
